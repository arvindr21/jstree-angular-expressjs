Add jsTree plugins

Steps :
  1. Download jsTree.directive.js OR run bower install jstree-directive
  2. Include the reference in your index.html
  3. Add 'jsTree.directive'as a dependency to your app and you can use the below plugins along with your tree

Add Checkbox plugin

Set tree-plugins="checkbox" 

<js-tree tree-plugins="checkbox" tree-data="json" tree-src="../jstree/data/tree-json.json"></js-tree>

Angular Directive Integration 
var app = angular.module('treeDemo', ['jsTree.directive'])

Add Custom Contextmenu plugin

Set tree-plugins="contextmenu"and pass the context menu object via tree-contextmenu="scopeObjName" 
<js-tree tree-plugins="contextmenu" tree-contextmenu="contextMenu" tree-data="json" tree-src="../jstree/data/tree-json.json"></js-tree>

Angular Directive Integration 
var app = angular.module('treeDemo', ['jsTree.directive'])


Scope object 
$scope.contextMenu = {
  show_at_node: false,
  items: {
    "Menu 1": {
      "label": "Menu 1",
      "action": function(obj) {
        console.log(obj);
        alert("You clicked " + obj.item.label);
      }
    },
    "Menu 2": {
      "label": "Menu 2",
      "action": function(obj) {
        console.log(obj);
        alert("You clicked " + obj.item.label);
      }
    }
  }
};

PS : As mentioned in the jsTree documentation, items is "an object of actions, or a function that accepts a node and a callback function and calls the callback function with an object of actions available for that node (you can also return the items too)."

Add Types plugin

Set tree-plugins="types"and pass the config object via tree-types="typesConfig" 

<js-tree tree-plugins="types" tree-types="typesConfig" tree-data="json" tree-src="../jstree/data/tree-json.json"></js-tree>

Angular Directive Integration 
var app = angular.module('treeDemo', ['jsTree.directive'])


Scope object 
$scope.typesConfig = {
      "default": {
        "icon": "/jstree/imgs/tree.png"
      },
      "demo": {
        "icon": "glyphicon glyphicon-ok"
      }
    }
